## Verification Items

For EACH signature tested, verify:

### Typography
- [ ] Font: Verdana applied (or fallback sans-serif)
- [ ] Size: Text is readable
- [ ] Weight: Bold headings, normal body
- [ ] Color: Text color matches design

### Layout
- [ ] Structure: Table layout intact
- [ ] Spacing: Padding/margins preserved
- [ ] Alignment: Elements align correctly
- [ ] Width: Fits in viewport

### Images
- [ ] Logo loads and displays
- [ ] Logo is correct size
- [ ] Logo is crisp (high-DPI)

### Links
- [ ] All links clickable
- [ ] Links open in new tab
- [ ] Correct URLs

### Dark Mode (if supported)
- [ ] Text readable in dark mode
- [ ] Dark logo variant displays
- [ ] Accent colors visible
