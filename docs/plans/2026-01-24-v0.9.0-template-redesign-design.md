## Overview

Complete redesign of all 6 email signature templates based on 2026 email signature best practices research. All templates redesigned from scratch with research-driven layouts and user-focused recommendations.

---

## Research Foundation

### 2026 Email Signature Best Practices

**Sources**:
- [Email Signature Design Trends for 2026 | Crossware365](https://www.crossware365.com/blog/email-signature-design-trends-for-2026-whats-in-and-whats-out)
- [What Is the Ideal Email Signature Size for 2026? | Mailmodo](https://www.mailmodo.com/guides/email-signature-size/)
- [WiseStamp Templates](https://www.wisestamp.com/templates/)
- [HubSpot Email Signature Generator](https://www.hubspot.com/email-signature-generator)

**Key Findings**:
- **Width**: 300-600px optimal (we chose 500px standard, 450px for Minimalist)
- **Height**: 150-200px recommended
- **Layout**: Left-aligned (not centered) - matches email text flow
- **Font**: Verdana (Zoho Mail default) - seamless integration with email body
- **Design Trend**: Minimalist, clean layouts with muted color palettes
- **Dark Mode**: Essential feature (already implemented in v0.6.0)
- **HTML-Rich**: Table-based layouts with inline styles (email client compatibility)

## Template Categories (Research-Driven)

Competitors (HubSpot, Canva, WiseStamp) categorize by **Design Style**, not layout or profession:
- Modern/Corporate
- Minimalist/Clean
- Creative/Bold
- Basic/Simple
- Compact

**Our 6 Categories**: Professional, Minimalist, Compact, Modern, Creative, Classic

**Default Template**: **Classic** (universal appeal, works for everyone)

## Content Hierarchy (3-Tier Structure)

All templates use this information architecture:

### Tier 1: Primary Contact (Direct Reach)
```
Phone • Email
```
**Font**: 12px Verdana
**Spacing**: 4px margin-bottom

### Tier 2: Personal Connections (Ways to Connect with YOU)
```
LinkedIn • X • Book a Meeting
```
**Font**: 12px Verdana
**Spacing**: 2px margin-bottom
**Note**: All 3 fields are **optional** in the main app (can show 0-3 links)

### Tier 3: Company Brand (Zoho Corporate Social)
```
Follow Zoho:
X • LinkedIn • Facebook • Instagram
```
**Font**: 11px Verdana
**Label**: 11px, #888888 (muted gray)
**Spacing**: 10px padding-top, 1px border-top (#E0E0E0)
**Note**: Entire section is **optional** (toggleable in main app)

**Rationale**: Clear visual hierarchy prevents personal social (your X/LinkedIn) from competing with Zoho brand accounts.

## Universal Design Specifications

### Typography
- **Font Family**: `Verdana, Geneva, sans-serif` (Zoho Mail default - NOT Arial)
- **Name**: 15-16px, bold, #333333
- **Title/Department**: 13px, #666666
- **Contact Links**: 12px
- **Social Links**: 11-12px

### Colors
- **Text Primary**: #333333 (name, primary info)
- **Text Secondary**: #666666 (title, department)
- **Accent Color**: Dynamic (Red/Green/Blue/Yellow) via inline styles
- **Separators**: Accent color bullets (•)
- **Social Label**: #888888 (muted, de-emphasized)

### Layout
- **Width**: 500px (all templates except Minimalist at 450px)
- **Height**: ~180-220px with full content
- **Alignment**: Left-aligned (NOT centered)
- **Structure**: HTML tables with inline styles (email client compatibility)

### Dark Mode
- **Implementation**: Dual-context CSS (preview mode vs email copy mode)
- **Logo Switching**: Light/dark logo variants with CSS display toggle
- **Text Colors**: White (#FFFFFF) for name, light gray (#E0E0E0) for title
- **Links**: Accent color (no override in dark mode)

### Accent Color Handling
- **Yellow Warning**: Display inline help message for yellow selection:
  > "⚠️ Yellow works best on dark backgrounds. For better visibility, consider Red, Green, or Blue."
- **Contrast-Aware**: Yellow uses dark text in Bold template (if name block is colored)

## Template Specifications

### 1. Classic Template (DEFAULT)

**Tag**: DEFAULT
**Description**: "Best for everyone. The standard email signature format that works across all roles, departments, and communication styles."

**Layout**:
```
[Logo - 34px height]

Name (15px bold)
Title | Department (13px)
Phone • Email (12px)
LinkedIn • X • Book a Meeting (12px)

─────────────── (1px border)
Follow Zoho: (11px, #888)
X • LinkedIn • Facebook • Instagram (11px)
```

**Structure**:
- Logo-top stacked layout (left-aligned)
- Vertical spacing: 10px logo-to-name, 7-8px between sections
- Logo links to zoho.com with UTM tracking
- Height: ~200-220px with full content

**Why Classic is Default**:
- Universal format recognized across all email clients
- Works for all roles, departments, and communication styles
- Safe, professional, timeless
- Logo-top is traditional but still left-aligned (2026 best practice)

## Implementation Notes

### Font Changes (CRITICAL)
- **OLD**: `font-family: Arial, Helvetica, sans-serif;`
- **NEW**: `font-family: Verdana, Geneva, sans-serif;`

**Rationale**: Verdana is Zoho Mail's default font. Signatures should match email body text for seamless integration.

### Logo URLs (Already Fixed in Repo)
```javascript
// Correct URL (fixed in earlier session)
'https://tejasgadhia.github.io/zoho-signature-generator/assets'
```

### Accent Color Implementation
- Use **inline styles**: `style="color: ${accentColor};"`
- Do NOT use `!important` for dynamic values (links)
- Keep `!important` ONLY for fixed values (name, title in dark mode)

### Dark Mode CSS Generation
```javascript
getDarkModeStyles(isPreview) {
  if (isPreview) {
    // Only .dark-mode class selectors (ignores system preference)
  } else {
    // Both @media (prefers-color-scheme: dark) AND .dark-mode class
  }
}
```

### Separator Bullets
- **Character**: `•` (middle dot, U+2022)
- **Color**: Accent color (inline style)
- **Spacing**: ` • ` (space-bullet-space)

### Known Limitations
- **Outlook Desktop**: May strip some CSS, fallback to inline styles
- **Gmail Mobile**: May adjust font sizes automatically
- **Dark Mode**: Only Gmail, Apple Mail, some Outlook versions support it

## Mobile Rendering Strategy

### Current Approach (v0.9.0)
- **Fixed 500px width** (450px for Minimalist)
- **Natural wrapping**: Let email clients handle narrow screens
- **No @media queries**: Most email clients don't support them well

### Expected Behavior
- **Desktop/Tablet (≥600px)**: Full 500px width, no wrapping
- **Phablets (375-600px)**: May require horizontal scroll or client zoom
- **Phones (320-375px)**: Likely requires horizontal scroll

### Future Work (Post-v0.9.0)
Create separate task for:
- [ ] Test on actual mobile email apps (Gmail iOS, Outlook Android, etc.)
- [ ] Consider responsive design with inline media queries
- [ ] Or create "Mobile-Optimized" template variants
- [ ] Reduce font sizes slightly for mobile (11-12px)
- [ ] Test Compact template on 320px screens (may wrap to 6+ lines)

**Priority**: Medium (not blocking v0.9.0 launch)

## Competitive Analysis Summary

### HubSpot (12 templates)
- Free, browser-based
- Mobile-first layouts, dark mode optimization
- Template 3 requires image removal for mobile (noted as limitation)

### Canva (Hundreds of templates)
- Design-focused, highly customizable
- Best for creative professionals
- May be overkill for simple corporate signatures

### WiseStamp (100+ templates)
- Categorized by style: Modern, Minimal, Creative, Simple, Basic, Clean, Compact
- Industry-specific recommendations
- Our approach mirrors their category system

### MySignature
- Free generator, carefully selected fonts, hundreds of icons
- Emphasis on simplicity and ease of use

**Our Differentiation**:
- Zoho-specific (brand colors, social accounts, bookings integration)
- Privacy-first (100% client-side, no tracking)
- 2026 best practices (Verdana font, left-aligned, minimalist trend)
- Research-driven design (not arbitrary aesthetics)
